Documentation de la Bibliothèque Mystical-Runic v0.5.2

Bienvenue dans la documentation de Mystical-Runic v0.5.2, un moteur de templating Rust haute performance avec fonctionnalités d'entreprise. Conçu pour la sécurité, les performances exceptionnelles, et l'intégration d'écosystème moderne. Plongez dans le monde de la magie du templating avec des symboles anciens pour des créations web modernes !

Fonctionnalités Clés v0.5.2

  • Sécurité Entreprise : Protection XSS par défaut, prévention de la traversée de chemins, classification des menaces et journalisation d'audit pour des applications web sûres.
  • Performance Optimisée : Compilation en bytecode, mise en cache des templates, rendu asynchrone, traitement concurrent et surveillance avancée des métriques.
  • Syntaxe de Templating Riche : Variables, conditionnelles, boucles imbriquées, inclusions récursives, héritage de templates, macros réutilisables et navigation d'objet profonde.
  • Filtres Puissants : Filtres intégrés (majuscules, minuscules, devise, mathématiques, date) avec chaînage et API flexible pour filtres personnalisés.
  • Internationalisation Avancée : Support multi-langues complet avec syntaxe {{t "key"}}, pluralisation intelligente et commutation de locale dynamique.
  • Outils de Développement : Messages d'erreur précis (ligne/colonne), débogage de templates, rechargement à chaud, intégration IDE complète (LSP, auto-complétion, coloration syntaxique).
  • Intégration Écosystème (v0.5.0+) : Support natif Axum, Warp, Actix-web, compatibilité WASM, outils CLI et traitement par lots.
  • Surveillance et Métriques : Collection avancée de métriques, monitoring des performances et optimisation avec OpenTelemetry.
  • Rust Moderne : Implémentation pure Rust, édition 2021, MSRV 1.74.0+, prêt pour Rust 2024, zéro dépendances superflues.
