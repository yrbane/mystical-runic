Enum TemplateValue (alias RuneSymbol)

TemplateValue est une énumération fondamentale dans Mystical-Runic. Elle représente les différents types de données que le moteur de template peut comprendre et manipuler. C'est la manière dont les données Rust natives (ou JSON) sont converties en un format utilisable par le moteur de rendu.

Définition


pub enum TemplateValue {
    String(String),
    Number(i64),
    Bool(bool),
    Array(Vec<TemplateValue>),
    Object(HashMap<String, TemplateValue>),
    // ... d'autres variantes peuvent exister pour des types spécifiques
}

Variantes

  • String(String) : Représente une valeur textuelle.
  • Number(i64) : Représente un nombre entier de 64 bits.
  • Bool(bool) : Représente une valeur booléenne (vrai/faux).
  • Array(Vec<TemplateValue>) : Représente une liste ordonnée de TemplateValues.
  • Object(HashMap<String, TemplateValue>) : Représente une collection de paires clé-valeur, où les clés sont des Strings et les valeurs sont des TemplateValues.

Création de TemplateValue

Vous pouvez créer des instances de TemplateValue directement :


use mystical_runic::TemplateValue;
use std::collections::HashMap;

let my_string = TemplateValue::String("Hello, Runic!".to_string());
let my_number = TemplateValue::Number(42);
let my_bool = TemplateValue::Bool(true);

let my_array = TemplateValue::Array(vec![
    TemplateValue::String("item1".to_string()),
    TemplateValue::Number(100),
    TemplateValue::Bool(false),
]);

let mut my_object_map = HashMap::new();
my_object_map.insert("product_name".to_string(), TemplateValue::String("Magic Wand".to_string()));
my_object_map.insert("price".to_string(), TemplateValue::Number(999));
let my_object = TemplateValue::Object(my_object_map);

Conversion depuis d'autres Types

Bien que vous puissiez construire TemplateValue manuellement, il est souvent plus pratique de convertir des types Rust courants ou des données JSON. La bibliothèque fournit des implémentations de From pour faciliter ces conversions.


use mystical_runic::TemplateValue;

let from_str: TemplateValue = "Hello".into(); // Convertit &str en TemplateValue::String
let from_i64: TemplateValue = 123i64.into(); // Convertit i64 en TemplateValue::Number
let from_bool: TemplateValue = true.into(); // Convertit bool en TemplateValue::Bool

// Pour Vec et HashMap, vous devrez peut-être construire les TemplateValue internes d'abord
