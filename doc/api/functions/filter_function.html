Type FilterFunction (alias MysticFilter)

FilterFunction est un type de fonction qui vous permet de définir des filtres personnalisés pour transformer les valeurs directement dans vos templates Mystical-Runic. Ces filtres sont appliqués en utilisant la syntaxe {{ variable | mon_filtre }} ou {{ variable | mon_filtre(arg1, arg2) }} si le filtre accepte des arguments. Ils sont un moyen puissant d'étendre les capacités de transformation de données du moteur.

Définition


pub type FilterFunction = Box<dyn Fn(&TemplateValue, Vec<TemplateValue>) -> TemplateResult<TemplateValue> + Send + Sync>;

C'est un alias de type pour une closure (fonction anonyme) qui doit implémenter les traits Fn, Send et Sync. Elle prend en entrée :

  • value: &TemplateValue : La valeur à filtrer.
  • args: Vec<TemplateValue> : Une liste d'arguments passés au filtre depuis le template.

Elle retourne un TemplateResult<TemplateValue>, qui est soit la valeur filtrée, soit une TemplateError en cas de problème (par exemple, type d'argument incorrect).

Création et Enregistrement d'un Filtre Personnalisé

Pour créer et utiliser un filtre personnalisé, vous devez définir une fonction ou une closure qui correspond à la signature de FilterFunction, puis l'enregistrer auprès de votre instance de TemplateEngine en utilisant la méthode register_filter.

Exemple : Filtre capitalize

Ce filtre convertit la première lettre d'une chaîne en majuscule. Il illustre comment accéder à la valeur et la transformer.


use mystical_runic::{TemplateEngine, TemplateContext, TemplateValue, TemplateResult, FilterFunction};

// Définition du filtre personnalisé
fn capitalize_filter(value: &TemplateValue, _args: Vec<TemplateValue>) -> TemplateResult<TemplateValue> {
    if let TemplateValue::String(s) = value {
        if let Some(first_char) = s.chars().next() {
            let capitalized = first_char.to_uppercase().to_string() + &s[first_char.len_utf8()..];
            Ok(TemplateValue::String(capitalized))
        } else {
            Ok(TemplateValue::String("".to_string())) // Chaîne vide reste vide
        }
    } else {
        // Retourne une erreur si le filtre est appliqué à un type non supporté
        Err(mystical_runic::TemplateError::RuntimeError("Filtre 'capitalize' attend une chaîne.".to_string()))
    }
}

fn main() -> TemplateResult<()> {
    let mut engine = TemplateEngine::new("templates");
    let mut context = TemplateContext::new();

    // Enregistrement du filtre avec le nom "capitalize"
    engine.register_filter("capitalize", Box::new(capitalize_filter));

    context.set("name", TemplateValue::String("alice".to_string()));

    let template = "Hello {{ name | capitalize }}!";
    let rendered = engine.render_string(template, &context)?;

    println!("{{}}", rendered); // Affiche: Hello Alice!

    Ok(())
}
