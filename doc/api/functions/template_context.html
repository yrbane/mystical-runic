Struct TemplateContext (alias RuneScroll)

TemplateContext est la structure fondamentale qui encapsule toutes les données que vous souhaitez rendre disponibles dans vos templates Mystical-Runic. Elle agit comme un conteneur clé-valeur, similaire à une carte (map) ou un objet JSON, où chaque clé correspond au nom d'une variable accessible dans le template, et chaque valeur est une instance de TemplateValue.

Création d'une Instance

TemplateContext::new()

Crée une nouvelle instance vide de TemplateContext. C'est le point de départ pour construire votre contexte de données.

  • Exemple : 
    
use mystical_runic::TemplateContext;

let mut context = TemplateContext::new();

Ajout de Données

set(&self, key: &str, value: TemplateValue)

Ajoute une paire clé-valeur au contexte. La clé est le nom de la variable tel qu'il sera utilisé dans le template, et la valeur est une instance de TemplateValue. Cette méthode est essentielle pour peupler votre contexte avec les données nécessaires au rendu.

  • Paramètres :
    • key : Le nom de la variable dans le template (par exemple, "user_name", "product_list").
    • value : La donnée à associer à cette clé, encapsulée dans un TemplateValue.
  • Exemple : 
    
use mystical_runic::{TemplateContext, TemplateValue};

let mut context = TemplateContext::new();
context.set("user_name", TemplateValue::String("Alice".to_string()));
context.set("age", TemplateValue::Number(30));
context.set("is_active", TemplateValue::Bool(true));

// Pour des objets imbriqués ou des tableaux, vous construisez des TemplateValue::Object ou TemplateValue::Array
let mut address_map = std::collections::HashMap::new();
address_map.insert("street".to_string(), TemplateValue::String("123 Main St".to_string()));
address_map.insert("city".to_string(), TemplateValue::String("Anytown".to_string()));
context.set("address", TemplateValue::Object(address_map));

Utilisation dans les Templates

Les données définies dans TemplateContext sont accessibles dans vos templates en utilisant la syntaxe de variable standard de Mystical-Runic. Pour les objets imbriqués, utilisez la notation par points.


<p>Nom: {{ user_name }}</p>
<p>Âge: {{ age }}</p>
{{#if is_active}}
    <p>Utilisateur actif.</p>
{{/if}}
<p>Adresse: {{ address.street }}, {{ address.city }}</p>
